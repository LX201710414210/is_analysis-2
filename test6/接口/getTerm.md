# 接口：getCourser  [返回](../README.md)
用例： [选择课程](../用例/选择课程.md)

- 权限：
    学生：能看到
    老师：可以看到

- 功能：
    返回所选择课程的当前学期。

- API请求地址：
   接口基本地址/v1/api/getTerm

- 请求方式 ：
    GET

- 请求参数说明:
    无

- 返回实例：

        {
         status：true
         term_id：1
        }

- 返回参数说明：

  |参数名称|说明|
  |:---------:|:--------------------------------------------------------|
  |status|bool类型，true表示正确的返回，false表示有错误|
  |term_id|返回学期数|
